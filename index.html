About APP

The APP is a simple JavaScript application, which offers to users a drawing functionality.

A short user guide

Workspace

Application contains three main sections:

  • Working area
  • Controls

Working area show the drawing space and basic information about the points, parallelogram and circle.

Usage

By using the APP, user is able to create parallelogram and circle only by choosing three points on the Working area. Simple! Just click the workarea three times ;)
